## shop_show

php自作

## 概要　

おすすめの飲食店を紹介できるサイトを作成しました。


## 機能

おすすめの飲食店を投稿、編集、削除できるようにしています。

よかった投稿に対していいねできたり、いいねした投稿を一覧できるようにしています。

パスワードリセットはメールアドレス宛に届くようにしています。
